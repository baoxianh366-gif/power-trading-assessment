# 电力交易潜质人才筛选测评系统

基于电力交易潜质评估模型开发的在线测评系统，用于筛选具备电力交易潜质的人才。

## 🌟 功能特点

### 测试人员端
- **登录注册**：支持新用户注册和继续未完成的测试
- **四模块测试**：
  - 模块一：数理与逻辑敏感度（5题，20分）
  - 模块二：风险与决策风格（4题，20分）
  - 模块三：模式识别与快速学习（3题，15分）
  - 模块四：日常决策偏好（24题，不计入总分）
- **防背题设计**：模块1-3各有5套题库，随机分配
- **计时功能**：每模块独立计时，超时自动提交
- **进度保存**：可随时暂停，下次继续

### 管理人员端
- **数据概览**：统计总测试人数、高潜质人才、平均分等
- **测试人员管理**：查看所有测试结果，支持搜索和筛选
- **潜质评估分析**：各维度平均分统计，潜质模型展示
- **面试指南**：筛选建议、关键题关注、面试问题推荐
- **数据导出**：支持导出测试数据和备份

## 📊 评分标准

| 评级 | 分数区间 | 说明 |
|------|---------|------|
| 高潜质 | ≥40分 | 优先安排面试 |
| 可培养 | 35-39分 | 具有培养价值 |
| 需考察 | <35分 | 需结合面试判断 |

*默认阈值，可在管理后台调整*

## 📱 技术栈

- **前端**：纯 HTML5 + CSS3 + JavaScript（无框架依赖）
- **数据存储**：localStorage（浏览器本地存储）
- **部署**：GitHub Pages（免费静态托管）

## ⚠️ 注意事项

1. 本系统使用浏览器本地存储，数据保存在用户浏览器中
2. 建议使用现代浏览器（Chrome、Firefox、Edge）访问
3. 管理员数据需在每台设备单独设置
4. 如需跨设备同步数据，请使用导出/导入功能

## 📝 题库说明

### 模块一：数理与逻辑敏感度（5套题库）
- 数字规律题
- 比例直觉题
- 逻辑推理题
- 期望值计算题
- 条件推理题

### 模块二：风险与决策风格（5套题库）
- 确定效应测试
- 止损判断
- 连续亏损后决策
- 信息顺序影响

### 模块三：模式识别与快速学习（5套题库）
- 规则应用题
- 反向规则学习
- 多因素推理题

### 模块四：日常决策偏好（固定24题）
- 风险偏好（4题）
- 时间偏好（2题）
- 分析倾向（4题）
- 学习速度（4题）
- 情绪恢复（4题）
- 矛盾检测（4题）
- 非理性认知（2题）
